cdfChiSquare

语法

cdfChiSquare(df, X)

参数

df 是正数，表示卡方分布的自由度。

X 是数值型标量或向量。

详情

返回卡方分布的累计密度函数的值。

例子

cdfChiSquare(1, [-1, 0, 0.5, 1, 2]);
# output
[0, 0, 0.5205, 0.682689, 0.842701]

cdfChiSquare(1, [0.1, 0.3, 0.5, 0.7, 0.9]);
# output
[0.24817, 0.416118, 0.5205, 0.597216, 0.657218]
