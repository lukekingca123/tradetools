asis

语法

asis(X)

参数

X 可以是任意数据类型。

详情

返回 X 的浅拷贝。

例子

a = 1 2 3
b = asis(a)
a[0] = 0
b
# output
[0, 2, 3]
b[1] = 4
# output
a;
[0, 4, 3]
