建库建表

关于此任务

通过 DolphinDB Web 交互界面创建一个按值（1 到 10）分区的数据库，并在该数据库中创建一个数据表。

过程

  1. 在 DolphinDB Web 的交互编程区域输入以下脚本后点击 执行。 
    CREATE DATABASE "dfs://firstdb"
PARTITIONED BY VALUE(1..10)

    由此，创建了名为 firstdb 的分布式数据库，该数据库按值（1 到 10）分区。

  2. 输入以下脚本后点击 执行。 
    CREATE TABLE "dfs://firstdb"."firsttb"(
    id INT,
    sym SYMBOL,
    price DOUBLE,
    qty INT
)
PARTITIONED BY id

    由此，在 firstdb 数据库中创建了名为 firsttb 的数据表。其中分区条件为 id。
