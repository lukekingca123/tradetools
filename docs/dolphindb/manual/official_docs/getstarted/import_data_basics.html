写入和查询

过程

  1. 执行以下脚本，生成模拟数据并写入分区表。 
    tmp = table(take(1..10,1000) as id, take(["A", "B", "C"],1000) as sym, rand(100.0,1000) as price, rand(1000,1000) as qty)
t = loadTable("dfs://firstdb","firsttb")
tableInsert(t,tmp)
  2. 执行以下脚本，查询分区表指定数据。
    • 查询分区表中所有数据条数：
      SELECT count(*) FROM t
    • 查询所有 id 为1的数据：
      SELECT * FROM t WHERE id=1
    • 根据 id 分组，查询每组内 price 列的平均值：
      SELECT avg(price) 
FROM t 
GROUP BY id
