C++ API

DolphinDB C++ API 用于连接 DolphinDB 服务端和 C++客户端，从而实现数据的双向传输和脚本的调用执行。 它可以方便您在 C++ 程序中使用 DolphinDB 进行数据的处理、分析和建模等操作，利用其优秀的计算性能和强大的存储能力来帮助您加速数据的处理和分析。在下文中提到的 API 均指代 DolphinDB C++ API。

C++ API Readme 包含以下五节：

  • 快速上手：介绍 API 的编译指南和简单示例。
  • 数据类型与格式：讲解 API 中 C++ 原生类型与 DolphinDB 中数据类型的相互转换，以及如何读写向量、表、字典等数据结构。
  • 基本操作：详细介绍 API 的基本用法，包括连接数据库、流订阅、数据写入等。
  • 实用工具类：介绍如何用内置的实用工具类来创建数据库、创建表格、执行查询语句等。
  • 故障排除：介绍常见的编译问题以及使用中可能遇到的异常，并给出解决办法。
