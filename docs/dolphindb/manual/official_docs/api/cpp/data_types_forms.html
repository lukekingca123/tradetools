数据对象

DolphinDB C++ API 中的数据对象包含标量、向量、矩阵、集合、字典和表等，其中：

  • Constant 类是所有类型的基类，所有的数据类型（例如 Int，String）和数据格式（例如 Vector，Table）都直接或间接派生自 Constant。

  • 以 SP 为结尾的类型（例如 ConstantSP，VectorSP）是一个智能指针类型，类似于标准库中的 std::shared_ptr，无需用户手动释放其内存。

  • 由上述两点可知，所有的数据对象都可以存储在 ConstantSP 中，可以根据其提供的接口 getType()和 getForm()来获取数据具体的类型和格式。

  • API 所有的类以及接口都在命名空间 dolphindb 中。
