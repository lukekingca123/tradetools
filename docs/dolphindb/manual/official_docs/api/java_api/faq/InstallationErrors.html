安装、使用错误

  • 问题描述：安装了 Java 但是报如下错误：'Windows 找不到文件 ‘javaw’。请确定文件名是否正确后，再试一次。'




  • 原因分析：Java 的路径没有配置到环境变量 path 中。
  • 解决办法：将Java 的路径添加到环境变量 path 中。注意，添加后须重新启动命令行，否则不会生效。
