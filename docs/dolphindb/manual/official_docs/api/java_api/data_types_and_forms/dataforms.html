数据形式

Java API 中支持 Scalar、Vector、Matrix、Set、Dictionary、Table、Tensor 七种数据形式，以下为详细介绍。

Scalar 接口（标量）

标量构造

在 API 中，所有的基本数据类型都实现了 Scalar 接口。以 BasicInt 为例，可以通过其构造方法声明一个标量：

Scalar scalar = new BasicInt(123);

标量基本使用

可以使用 getNumber 方法或者 getString 方法来获取基本数据类型的值。

以 BasicInt 为例，其获取基本数据类型的值方式如下：

BasicInt basicInt = new BasicInt(36);
String basicIntString = basicInt.getString();
Number basicIntNumber = basicInt.getNumber();

注意：getNumber 方法仅能返回数字类型，对于例如 BasicString 这样的非数字类型会抛出异常。示例如下：

Scalar scalar = new BasicString("123123");
System.out.println(scalar.getNumber());

会抛出如下异常：

'java.lang.Exception: Incompatible data type'

目前 Java API 中支持 getNumber 方法的数据类型有：

BasicBoolean、BasicByte、BasicComplex、BasicDecimal32、BasicDecimal64、BasicDecimal128、BasicDouble、BasicDuration、BasicFloat、BasicInt、BasicInt128、BasicLong、BasicPoint、BasicShort、BasicString、Void。

Vector（向量）

API 提供各种数据类型的 Vector 形式，此 Vector 并非 Java 自带的 Vector，它在 *com.xxdb.data *包下，下面例子中将以 BasicIntVector 举例（其他类型的 Vector 同理）。

向量构造

BasicIntVector 构造函数如下：

// 通过容量来定义。
public BasicIntVector(int size)

// 通过list来定义，定义后vector有初始值，即为list中的内容。
public BasicIntVector(List<Integer> list)

使用向量

  1. 通过 add 方法来向 Vector 尾部添加值。

    BasicIntVector v = new BasicIntVector(6);
v.add(1);
v.add(2);
v.add(3);
for (int i=0; i<=v.rows(); i++) {
   System.out.print(v.get(i)+" ");
}

/*
运行结果：
0 0 0 0 0 0 1 2 3 0 
* 可以看到是在第7个下标处进行添加操作，而非从头开始添加
*/

  2. 通过setInt方法修改向量的值，示例代码如下：

    BasicIntVector v = new BasicIntVector(6);
v.setInt(0,2147483647);
v.setInt(1,-2147483647);
v.setInt(2,99);
v.setInt(3,0);
v.setInt(4,-12);

    其中setInt定义如下：

    public void setInt(int index, int value)

    参数介绍

    index 表示要修改元素的下标，从 0 开始计数。

    value 表示修改元素的值，必填参数。

  3. 通过 get 方法来获取 Vector 某一个下标的值：

    // 申明一个大小为5的Vector
BasicIntVector v = new BasicIntVector(6);
v.setInt(0,2147483647);
v.setInt(1,-2147483647);
v.setInt(2,99);
v.setInt(3,0);
v.setInt(4,-12);
// 获取vector所有的行数，注意，这里获取的值为6，而不是5,因为定义的Vector大小为6，没有设置的部分会以0或者null填充
for (int i=0; i<v.rows(); i++) {
    System.out.println(v.get(i));
}

返回 Vector 的查询语句

在 DolphinDB 中，如果查询语句以 exec 开头、且查询的结果只有一列的话，其返回值会是一个 Vector，而不是 Table。下例中会返回一个 Vector：

DBConnection conn = new DBConnection();
conn.connect(HOST,PORT,USER,PASSWORD);
String script = "exec x from loadTable(\"dfs://ramd\",`pt1)";
Vector vector = (Vector) conn.run(script);

如果用 Table 对象去接收返回值会抛出以下异常：

// 代码
Table table = (Table) conn.run(script);
// 报错
java.lang.ClassCastException: com.xxdb.data.BasicDoubleVector cannot be cast to com.xxdb.data.Table

Matrix 接口（矩阵）

矩阵构造

Matrix 即矩阵，是一个接口，由各种基本数据类型的矩阵形式来实现。以 BasicIntMatrix 为例，其构造方法如下：

public BasicIntMatrix(int rows, int columns, List<int[]> list)

参数介绍

rows 矩阵行大小。

column 矩阵列大小。

list 表示矩阵每个位置的初始值，可选参数。

以下为声明一个两行两列的 BasicIntMatrix 类型的矩阵具体例子：

int[] row1 = {1,2};
int[] row2 = {3,4};
BasicIntMatrix matrix = new BasicIntMatrix(2, 2, Arrays.asList(row1, row2));

矩阵基本使用

  1. 可以使用 get 方法获取矩阵中的某一处（用行列坐标确定）的值，返回值为一个标量（Scalar）：

    public Scalar get(int row, int column)

  2. 可以使用 set<data_type> 方法（其中<data_type>代表数据类型，例如数据类型为 Int，则方法为 setInt）设置矩阵某处的值，以 setInt 为例：

    // 例如，设置矩阵第1行第2列的值为3
matrix.setInt(1, 2, 3);

Set 接口（集合）

Java API 提供 Set 集合接口，可以用于存储一系列相同类型的元素，如整数等。API 中提供了 Set 接口来构造和使用一个集合，具体实现类为 BasicSet。

集合构造

集合的元素称为 key，BasicSet 集合的构造方法如下：

public BasicSet(DATA_TYPE keyType, int capacity)

参数介绍

keyType 表示声明集合的类型，枚举类（枚举内容见附件1）。

capacity 表示集合的初始容量，可选参数，默认为 0。BasicSet 负载因子为 0.75，当元素数量达到负载因子乘以当前容量时，BasicSet 会自动进行扩容（二倍扩容）。

集合使用

  1. Set 使用 add 方法来向集合中添加对象，例如：

    BasicSet basicSet = new BasicSet(Entity.DATA_TYPE.DT_INT,4);
basicSet.add(new BasicInt(1));

  2. Set 可以使用 keys 方法来获取所有的 key，方法如下：

    public Vector keys();

    其方法返回一个 Vector（注意是 com.xxdb.data 包下的 Vector，并非 Java 自带的 Vector），此时可以通过这个 Vector 获取所有的集合元素，使用方法如下：

    BasicSet basicSet = new BasicSet(Entity.DATA_TYPE.DT_INT);
basicSet.add(new BasicInt(1));
basicSet.add(new BasicInt(2));
basicSet.add(new BasicInt(3));
basicSet.add(new BasicInt(4));
basicSet.add(new BasicInt(5));
Vector vector = basicSet.keys();
System.out.println(vector.getString());

  3. Set 可以通过 contains 方法来判断一个 key 是否存在与集合中，其返回一个 boolean 类型的值，示例代码如下：

    BasicSet basicSet = new BasicSet(Entity.DATA_TYPE.DT_INT,4);
basicSet.add(new BasicInt(1));
basicSet.add(new BasicInt(2));
basicSet.add(new BasicInt(3));
basicSet.add(new BasicInt(4));
basicSet.add(new BasicInt(5));
System.out.println(basicSet.contains(new BasicInt(4)));

Dictionary（字典）

字典是一种常用的数据形式，采用键值对的方式来存储一组关系的映射。

字典构造

构造方法如下：

public BasicDictionary(DATA_TYPE keyType, DATA_TYPE valueType, int capacity)

参数介绍

keyType 表示字典中键（Key）的数据类型，为一个枚举（枚举内容见附件1）。

valueType 表示字典中值（Value）的数据类型，为一个枚举（枚举内容见附件1）。

capacity 表示初始字典的大小，可选参数，默认值为0。当存储的键值对超过 capacity 时，字典会自动进行扩容。

使用字典

  1. 可以通过 put 方法向 BasicDictionary 中存放一个键值对：

    BasicDictionary basicDictionary = new BasicDictionary(Entity.DATA_TYPE.DT_STRING, Entity.DATA_TYPE.DT_INT);
// 例如，键类型：BasicString 值类型：BasicInt
basicDictionary.put(new BasicString("1"),new BasicInt(2));

  2. 可以通过 get 方法从 BasicDictionary 中获取一个键对应的值：

    BasicDictionary basicDictionary = new BasicDictionary(Entity.DATA_TYPE.DT_STRING, Entity.DATA_TYPE.DT_INT);
basicDictionary.put(new BasicString("1"),new BasicInt(2));
System.out.println(basicDictionary.get(new BasicString("1")));

/*
运行结果：
2
*/

  3. 可以通过 keys 方法查询 BasicDictionary 中所有的键。

    注意：与 BasicSet 不同，BasicDictionary 的keys方法返回值为一个泛型为 Entity 的 Set，示例代码如下：

    BasicDictionary basicDictionary = new BasicDictionary(Entity.DATA_TYPE.DT_STRING, Entity.DATA_TYPE.DT_INT);
basicDictionary.put(new BasicString("1"),new BasicInt(1));
basicDictionary.put(new BasicString("2"),new BasicInt(2));
basicDictionary.put(new BasicString("3"),new BasicInt(3));
Set<Entity> entities = basicDictionary.keys();
for (Entity e : entities) {
    System.out.print(e.getString() + " ");
}

/*
  运行结果：
  1 2 3 
*/

  4. 可以通过 getString 方法直接获取 BasicDictionary 中的映射关系：

    BasicDictionary basicDictionary = new BasicDictionary(Entity.DATA_TYPE.DT_STRING, Entity.DATA_TYPE.DT_INT);
basicDictionary.put(new BasicString("1"),new BasicInt(4));
basicDictionary.put(new BasicString("2"),new BasicInt(5));
basicDictionary.put(new BasicString("3"),new BasicInt(6));
System.out.println(basicDictionary.getString());

/*
  运行结果：
  {1,2,3}->{4,5,6}
*/

Table 接口（表）

API 中定义了 Table 的数据形式以方便用户对数据的查询与使用。

表构造

BasicTable 构造函数如下：

public BasicTable(final List<String> colNames, final List<Vector> cols)

参数介绍

colNames 表示表的列名。

cols 表示每一列的具体数据。

通过查询获取表数据

DolphinDB 中大多 SQL 查询语句的结果为一张表，因此可以通过执行查询语句来直接获得一张表，示例代码如下：

DBConnection dbConnection = new DBConnection();
DBConnection dbConnection = new DBConnection();
dbConnection.connect(HOST,PORT,USER,PASSWORD);
BasicTable basicTable = (BasicTable) dbConnection.run("select * from loadTable(\"dfs://ramd\",`pt3)");

使用表

  1. 通过 getColumn 方法获取某一列的所有值，方法定义如下：

    // 第一种定义，可以通过列号来获取
public Vector getColumn(int index)

// 第二种定义，可以通过列名来获取
public Vector getColumn(String name)

    其返回值是一个 Vector 对象（com.xxdb.data.Vector），可以通过这个 Vector 对象来遍历所有数据。

  2. 直接操作 getColumn 返回的 Vector 对象来达到修改表中的内容的目的，示例代码如下：

    public void test_set_nullvalue() throws IOException {
  try{
      List<String> colNames =  Arrays.asList("cint","cdouble","clong","cfloat","cshort");
      List<Vector> colData = Arrays.asList(new BasicIntVector(1),new BasicDoubleVector(1),new BasicLongVector(1),new BasicFloatVector(1),new BasicShortVector(1));
      BasicTable bt = new BasicTable(colNames,colData);
      bt.getColumn("cint").set(0,new BasicInt(Integer.MIN_VALUE));
      bt.getColumn("cdouble").set(0,new BasicDouble(-Double.MIN_VALUE));
      bt.getColumn("clong").set(0,new BasicLong(Long.MIN_VALUE));
      bt.getColumn("cfloat").set(0,new BasicFloat(-Float.MAX_VALUE));
      bt.getColumn("cshort").set(0,new BasicShort(Short.MIN_VALUE));
      bt.getColumn("cshort").set(0,new BasicShort(Short.MIN_VALUE));
      bt.getColumn("cshort").set(0,new BasicShort(Short.MIN_VALUE));
      assertTrue(((Scalar)bt.getColumn("clong").get(0)).isNull());
  }catch (Exception e){
      e.printStackTrace();
  }
}

  3. 如果想直接生成每一行 Json 格式的数据，可以调用 getRowJson 方法来直接生成每一列 Json 格式的数据。具体代码如下：

    DBConnection dbConnection = new DBConnection();
dbConnection.connect(HOST,PORT,USER,PASSWORD);
BasicTable basicTable = (BasicTable) dbConnection.run("select * from loadTable(\"dfs://ramd\",`pt3)");
// 获取行数
int rows = basicTable.rows();
// 遍历每一行
for(int i=0;i<rows;i++){
    System.out.println(basicTable.getRowJson(i));
}

Java API 提供部分数据类型的 Tensor 数据形式：

DT_BOOL
DT_CHAR
DT_SHORT
DT_INT
DT_LONG
DT_FLOAT
DT_DOUBLE

Java API 提供 BasicTensor 类来获取构造后的 Tensor。注意，目前仅支持通过运行 DolphinDB 的函数 tensor() 来构造 Tensor 数据，暂时不能通过 BasicTensor 构造。

使用示例

@Test
public void test_tensor_1() throws IOException {

    // Java api 里可以使用 BasicTensor 来进行 tensor 数据的接受
    BasicTensor tensor = (BasicTensor) conn.run("tensor(((1..1000$50:20),(1..1000$50:20)))");
    // 对 tensor 数据进行打印展示
