登录 login

除了上一节介绍介绍的使用 connect 方法输入指定参数进行登录外，用户还可使用 login 方法进行登录。以下介绍 login 方法及其参数。

以下为使用登录方法 login 的完整语法。

public void login(String userID, String password, boolean enableEncryption)

参数介绍

userID String 类型，表示登录的用户名。

password String 类型，表示登录的密码。

enableEncryption boolean 类型，表示登录时是否采用加密。若开启加密，加密模式将采用公钥加密，具体参考 login 函数介绍。

示例

下例代码将以 userId 为"admin", password为 "123456"的身份进行登录，同时开启加密。

@Test
public void testLogin() throws IOException {
    DBConnection dbConnection = new DBConnection();
    dbConnection.connect("192.168.1.167", 18921);
    dbConnection.login("admin", "123456", true);
}
